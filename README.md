beetmachine
===========

An HTML5 Music Sampler/Sequencer


About
-------------
A single-page app for building patterns/sequences, sampling and mixing sound projects entirely through the browser.
Use some of the pre-loaded sounds attached to the pads, or drag a wav or mp3 file onto a pad to change it's sample.

Technos
-------------
beetmachine is developed in a [yeoman](https://github.com/yeoman/yeoman) workflow with [require.js](https://github.com/jrburke/requirejs), [backbone.js](https://github.com/jashkenas/backbone) and the amazing [timbre.js](https://github.com/mohayonao/timbre.js)
